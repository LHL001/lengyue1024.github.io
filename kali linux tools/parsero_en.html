Parsero Package Description

Parsero is a free script written in Python which reads the Robots.txt file of a web server and looks at the Disallow entries. The Disallow entries tell the search engines what directories or files hosted on a web server mustn‚Äôt be indexed. For example, ‚ÄúDisallow: /portal/login‚Äù means that the content on www.example.com/portal/login it‚Äôs not allowed to be indexed by crawlers like Google, Bing, Yahoo‚Ä¶ This is the way the administrator have to not share sensitive or private information with the search engines.

Parsero will check the HTTP status code in the same way for each Bing result.
